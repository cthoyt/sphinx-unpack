Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/sphinx_unpack>`_ with uv:

.. code-block:: console

    $ uv pip install sphinx_unpack

or with pip:

.. code-block:: console

    $ python3 -m pip install sphinx_unpack

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/sphinx-unpack.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/sphinx-unpack.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/sphinx-unpack.git
    $ cd sphinx-unpack
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
